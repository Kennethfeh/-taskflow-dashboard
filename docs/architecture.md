## Journey
1. Hero sets context and shows logged-in UI.
2. Swimlanes communicate how work moves.
3. Blueprint route covers IA + dev handoff.
4. Docs provide token tables + notes.

## Wireframe Guidance
- 12-column layout, 1120px max width.
- 8px baseline grid, 24px padding on cards.
- Buttons: 48px min height, 24px horizontal padding.
